# Politica de Seguranca

## Versoes Suportadas

| Versao | Suportada          |
| ------ | ------------------ |
| 1.x.x  | :white_check_mark: |
| < 1.0  | :x:                |

## Reportando uma Vulnerabilidade

Se voce descobrir uma vulnerabilidade de seguranca, por favor:

1. **Nao** abra uma issue publica
2. Envie um email para o mantenedor com detalhes da vulnerabilidade
3. Inclua:
   - Descricao da vulnerabilidade
   - Passos para reproduzir
   - Impacto potencial
   - Sugestoes de correcao (se houver)

## Tempo de Resposta

- Confirmacao de recebimento: 48 horas
- Avaliacao inicial: 7 dias
- Correcao (se confirmada): 30 dias

## Escopo

Esta politica cobre:

- Codigo fonte do FogStripper
- Dependencias diretas do projeto
- Scripts de instalacao e configuracao

## Fora do Escopo

- Vulnerabilidades em dependencias upstream (reporte diretamente aos mantenedores)
- Ataques de engenharia social
- Ataques fisicos

## Reconhecimento

Contribuidores que reportarem vulnerabilidades validas serao reconhecidos no CHANGELOG (se desejarem).
